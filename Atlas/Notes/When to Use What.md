> When to use Obsidian, when to use Claudian, when to use Cowork, and what to ask.
## Obsidian Is For

- **Reading and browsing notes** — click through the folder tree, follow wikilinks, explore the graph
- **Editing notes directly** — fix a typo, add a thought, leave `%%inline comments%%` for Claude
- **Creating daily notes** — click today's date in the calendar widget
- **Visual navigation** — the graph view shows how your notes connect
- **Quick capture** — Cmd+N to create a new note, jot something down, move on
## Claudian Is For (Obsidian Sidebar)

Everything that involves thinking, processing, or creating across your notes — without leaving Obsidian. Claudian is a sidebar chat panel that uses Claude Code under the hood. It can read, write, edit, and search your vault directly.

## Cowork Is For (Outside Obsidian)

Cowork in Claude Desktop gives you the same Claude AI but outside Obsidian. Use it when your work doesn't start from a note — research sessions, brainstorming, or tasks that span multiple apps.

Cowork reads the same CLAUDE.md and understands your vault structure. To set it up: open Claude Desktop → Cowork tab → select your vault folder.
## The Key Principle

**Obsidian = reading and light editing. Claudian = in-Obsidian AI. Cowork = outside-Obsidian AI.**

If you find yourself manually filing, formatting, linking, or organizing notes in Obsidian, stop. Ask Claude to do it instead. That's the whole point of IKE — AI handles the maintenance so your system doesn't decay.

> When to use Obsidian, when to use Claude, and what to ask.
## Obsidian Is For

- **Reading and browsing notes** — click through the folder tree, follow wikilinks, explore the graph
- **Editing notes directly** — fix a typo, add a thought, leave `%%inline comments%%` for Claude
- **Creating daily notes** — click today's date in the calendar widget
- **Visual navigation** — the graph view shows how your notes connect
- **Quick capture** — Cmd+N to create a new note, jot something down, move on
## Claude Is For

Everything that involves thinking, processing, or creating across multiple notes.
### Day One Conversations

These work immediately with the starter vault and Cowork:

| What you want | What to say to Claude |
|---|---|
| Plan your day | "What should I work on today?" |
| Create a daily note | "Create today's daily note with my meetings" |
| Create a note | "Create a note about [topic]" |
| Process inline comments | "Review the comments in [note name]" |
| Summarize a note | "Summarize [note name]" |
| Research a topic | "Research [topic] and create a note" |
| Draft an email | "Draft an email to [person] about [topic]" |
| Create a Map of Content | "Create a Map of Content for [topic]" |
| Format a web clip | "Format the web clip in +/Web/[note]" |
| Continue working on a project | "Let's keep working on [effort name]" |
| File inbox items | "Process the inbox" |
### With Connectors (Gmail, Calendar, Drive)

| Draft a reply | "Draft a reply to [person's] email about [topic]" |
| Check your schedule | "What's on my calendar today?" |

### As Your Vault Grows

Over time, Claude becomes more useful as your vault fills with context:

| What you want | What to say to Claude |
|---|---|
| Synthesize across meetings | "What themes are emerging from this week's meetings?" |
| Track decisions | "What decisions have we made about [topic]?" |
| Prepare for a meeting | "Pull together everything I know about [topic] for my meeting" |
| Find connections | "What notes relate to [topic] that I might have missed?" |
| Weekly review | "Create a weekly review for this week" |
## The Key Principle

**Obsidian = reading and light editing. Claude = everything else.**

If you find yourself manually filing, formatting, linking, or organizing notes in Obsidian, stop. Ask Claude to do it instead. That's the whole point of IKE — AI handles the maintenance so your system doesn't decay.
